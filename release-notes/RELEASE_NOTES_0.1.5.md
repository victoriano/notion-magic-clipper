## 📝 Installation (unpacked)
1. Unzip the `notion-magic-clipper-0.1.5.zip` archive.
2. Open `chrome://extensions/` in Chrome.
3. Enable Developer mode.
4. Click "Load unpacked" and select the unzipped folder.
